# 💫 About Me:
Hey there! 👋 I'm an iOS-focused Software Engineer with 3+ years of experience shipping production-grade mobile and web apps using Swift, SwiftUI, React.js, and Salesforce.

These days I'm pursuing my Master's in Informatics Engineering at Universidad de Oviedo in Spain, and I specialize in iOS development, API integrations, Salesforce development, and full-stack web solutions — always looking for ways to build smarter and ship better.

A few things I'm proud of:

- 📱 Built production-grade iOS features using Swift, SwiftUI, and UIKit for solutions used by 100+ museum and art gallery clients.

- 🔌 Designed and maintained REST API integrations between Salesforce and third-party platforms in production environments.

- 🛍️ Built responsive, SEO-optimized web platforms with React.js, contributing to measurable organic traffic growth.

- ✅ Worked with XCTest and Apex test classes to maintain strong code quality and production stability.

⚡️ Fun Fact: I'm a huge fan of trying out new apps, testing innovative digital products, and being a beta tester for emerging tech—I love seeing how things work (and how they can be improved)!

🌐 Want to see what I've been building? Check out my portfolio at **[ethanmont.dev](https://www.ethanmont.dev)** — would love to hear what you think!

I'm open to **iOS Developer** and **Software Engineer** roles across Europe — if you're working on something exciting, let's talk! 🚀
